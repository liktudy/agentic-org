# agentic-org Claude Code Entry

This repository is a headquarters-centered multi-agent organization system.

Use these files as the source of truth:

- `AGENTS.md`
- `ORG_STRUCTURE.md`
- `docs/AI_READING_GUIDE.md`
- `docs/VISION.md`
- `docs/ARCHITECTURE.md`
- `docs/OPERATING_MANUAL.md`
- `departments/governance/policies/`
- `departments/governance/workflows/`
- `departments/organization-design/policies/`
- `departments/workflow-factory/workflows/`
- `departments/workflow-factory/prompts/`
- `companies/*/company.spec.json`
- `companies/*/ORG_STRUCTURE.md`
- `companies/*/WORKFLOW.md`
- `companies/*/departments/*/agents/*.md`

The Highest Leader is the final decision maker. Reuse existing companies before creating new ones. Company creation and company execution are separate decisions. Standards follow `departments/governance/policies/STANDARD_LEVEL_POLICY.md`. Boundaries follow `departments/governance/policies/BOUNDARY_POLICY.md`. Text source files must use UTF-8 encoding. `agentic-org` is an English-only execution project. AI/agent tools must not load the separate `agentic-org-zh` project unless Chinese explanation, review, editing, or consistency checks are requested. Internal standards and generated artifacts must be written in English first. Chinese companion material may be generated later in `agentic-org-zh`. Organization changes must follow `departments/governance/policies/ORGANIZATION_CHANGE_POLICY.md`. New role agents must follow `departments/governance/templates/role-agent-template.md`. Modifications and deletions require rollback data before execution.
