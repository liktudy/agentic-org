# agentic-org Claude Code Entry

This repository is a headquarters-centered multi-agent organization system.

Use these files as the source of truth:

- `AGENTS.md`
- `ORG_STRUCTURE.md`
- `docs/VISION.md`
- `docs/ARCHITECTURE.md`
- `docs/OPERATING_MANUAL.md`
- `departments/governance/`
- `departments/organization-design/`
- `departments/workflow-factory/`
- `companies/*/company.spec.json`
- `companies/*/ORG_STRUCTURE.md`
- `companies/*/WORKFLOW.md`
- `companies/*/departments/*/agents/*.md`

The Highest Leader is the final decision maker. Reuse existing companies before creating new ones. Company creation and company execution are separate decisions. Standards follow `departments/governance/policies/STANDARD_LEVEL_POLICY.md`. Organization changes must follow `departments/governance/policies/ORGANIZATION_CHANGE_POLICY.md`. Modifications and deletions require rollback data before execution.
