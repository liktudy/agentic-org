# Output Artifact Standards

## 中文版

### 内部产物

子公司内部正式产物只允许使用：

- Markdown: `.md`
- CSV: `.csv`
- JSON: `.json`

内部产物用于任务推进、审核、数据沉淀和集团交付部加工。

### 外部交付物

外部交付物由集团交付部生成。可包含：

- PPT / PPTX
- Word / DOCX
- PDF
- HTML
- Markdown
- CSV
- JSON

### 路径

子公司内部产物：

```text
companies/<company>/outputs/internal/tasks/<task-id>/
```

集团外部交付物：

```text
departments/delivery/outputs/<company>/drafts/
departments/delivery/outputs/<company>/final/
departments/delivery/outputs/<company>/archive/
```

`outputs/` 目录不提交 Git。

## English Version

Company internal artifacts are limited to `.md`, `.csv`, and `.json`. External deliverables are generated by the Delivery Department.
